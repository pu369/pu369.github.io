利用github免费空间

   若只需本地代码调试，可跳过本页面内容。
   利用github可以免费建立静态网站。
   天朝访问github需科学上网（shadowsocks 、VPN 等）以及安装git并在cmd中执行: 

set https_proxy=127.0.0.1:1080
set http_proxy=127.0.0.1:1080
set GIT_SSL_NO_VERIFY=true

   1、在github注册用户名、密码。
   2、比如我的用户名是pu369,就在github上建立名为pu369.github.io的repositorie
   3、打开cmd命令窗口，输入 

cd c:\

   4、再输入以下命令，将github上库clone到本地，即：生成C:\pu369.github.io目录。 

git clone  https://github.com/pu369/pu369.github.io.git

   5、将C:\pu369.github.io目录中除.git目录外，其他的全删除，并新建index.html文件 ，内容为hello world
   6、如果你有顶级域名，如www.your.com,可以在C:\pu369.github.io目录中再新建一个名为CNAME的文件 （无扩展名），内容为your.com ,则会将域名绑定到你的github网站。
   7、将C:\pu369.github.io目录名改名为www，并替换c:\wamp\www目录。
   8、好了，以后每次在c:\wamp\www目录中新建或修改文件后，都从cmd命令窗口转到c:\wamp\www目录，然后执行以下代码来更新github上的内容（根据提示输入githhub用户名和密码）： 

git add -A
git commit -a -m "hello"	
git push
