# #欢迎GitHub页面。

本页面自动发电机是为所有人创造你的项目页面漂亮的最简单的方法。作者网页内容使用GitHub味降价，选择一个模板的设计制作，出版。生成后你的页面，你可以检查出新的分支： $ cd your_repo_root / repo_name $ git fetch origin $ git checkout GH页 如果你使用GitHub为MAC，简单地同步你的版本库，你会看到新的分支。

# #设计师模板

我们已经制定了一些漂亮的模板供你使用。继续继续布局浏览。你可以很容易地回到之前出版编辑您的网页。发布你的页面后，您可以重新页发生器和切换到另一个主题。你的页面内容将如果它依然Markdown格式保存。

# #而驱动棒？

如果你不喜欢使用自动生成器，推动一个分支命名为GH页你的库来创建一个网页，手动。除了支持常规的HTML内容，GitHub页面支持博士，一个简单的静态网站，博客了解发电机由我们自己的汤姆Preston沃纳写的。哲基尔可以很容易的创建站点范围的页眉和页脚不复制他们在每一页。它还提供智能化的博客支持和其他先进的模板的特征。

# #作者和贡献者

你可以生成一个链接到文件提到一个GitHub的用户名。由此产生的<一>元素将贡献者链接的GitHub剖面。例如：2007，克里斯wanstrath（@ defunkt），普Hyett（@ pjhyett），和汤姆沃纳（@ mojombo）成立GitHub。
