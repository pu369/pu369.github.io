git clone 命令克隆远程仓库代码到本地（win7系统）

首先得会科学上网，安装git,配置代理，然后在github上找到远程仓库地址，在cmd中执行以下命令，将远程库代码clone到当前目录下：

	git clone
	https://github.com/shadowsocks/shadowsocks-android.git

git commit和git push 命令提交修改后的代码到远程仓库（win7系统）

在cmd中，转到自己的代码仓库听本地分支下，依次执行：

git add -A
git commit -a -m "hello"	
git push

根据提示，输入你的github用户名和密码，完成更新代码。
