win7下搜索文件内容的办法

目录下有静态html文件，想根据关键词搜索，决定用批处理： 在该目录下建search.bat：

for %%i in (*.html) do find "要搜索的关键词" %%i
pause

更多批处理用法可参考

www.zybuluo.com/yangfch3/note/338252


git clone 命令克隆远程仓库代码到本地（win7系统）

首先得会科学上网，安装git,配置代理，然后在github上找到远程仓库地址如： https://github.com/shadowsocks/shadowsocks-android.git 在cmd中执行以下命令，将远程库代码clone到当前目录下：

git clone https://github.com/shadowsocks/shadowsocks-android.git

git commit和git push 命令提交修改后的代码到远程仓库（win7系统）

在自己的代码仓库听本地分支下执行：

git commit -a -m "hello"
	     git push

根据提示，输入你的github用户名和密码，完成push。


如何克隆DOM节点

script：

window.onload = function () { 
var sourceNode = document.getElementById("div-0"); // 获得被克隆的节点对象 
for (var i = 1; i < 10; i++) { 
var clonedNode = sourceNode.cloneNode(true); // 克隆节点 
clonedNode.setAttribute("id", "div-" + i); // 修改一下id 值，避免id 重复 
sourceNode.parentNode.appendChild(clonedNode); // 在父节点插入克隆的节点 
} 
}
